devWM
=====

devWM is a very simple and lightweight tiling window manager based on catwm.

Modes
-----

    --------------   ----------------   ----------------
    |        |___|   |              |   |              |
    |        |___|   |    Master    |   |              |
    | Master |___|   |              |   |    Master    |
    |        |___|   |--------------|   |              |
    |        |___|   |    |    |    |   |              |
    --------------   ----------------   -----------------


Installation
------------

Need Xlib, then:
    $ vim config.h
    $ make
    # make install
    $ make clean
